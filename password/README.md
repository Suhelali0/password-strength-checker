ShieldPass: Advanced Password Strength Checker & Generator
ShieldPass ek modern web application hai jo users ko unke passwords ki strength check karne aur highly secure, random passwords generate karne mein madad karta hai. Ise specifically internships aur portfolio building ke liye ek clean Glassmorphism UI ke saath design kiya gaya hai.
